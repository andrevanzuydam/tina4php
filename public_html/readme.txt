TINA4PHP - This is not a framework for PHP

This features a RAD tool for PHP with version and release control.
Cool stuff like coffeescript parsing and SASS.  
In browser editor and most database support with CDE.
This tool is still in development and has written 10 commercial applications.

INSTALL

Double Click on the mongoose.exe file to run a webservice

Login to the http://localhost:7070/?interface=Developer with admin & admin

Browse to the http://localhost:7070/ to see what is going on
