TINA4PHP - This is not a framework for PHP

This features a RAD tool for PHP with version and release control.
Cool stuff like coffeescript parsing and SASS.  
In browser editor and most database support with CDE.

INSTALL

Double Click on the mongoose.exe file to run a webservice, it should work but we have experienced problems with Internet Explorer 11.

Login to the http://localhost:7007/?interface=Developer with admin & admin

Browse to the http://localhost:7007/ to see what is going on
